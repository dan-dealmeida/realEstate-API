# API Endpoints

## Favorites
Endpoints pertaining to favorites

#### GET /favorites
- Retrieves a paginated list of all favorites

#### POST /favorites
- Adds a new favorite

#### PUT /favorites/{id}
- Updates an existing favorite

#### DELETE /favorites/{id}
- Deletes an existing favorite

## RealEstate
Endpoints concerning real estate

#### GET /realEstate
- Retrieves a paginated list of all real estate properties

#### POST /realEstate
- Creates a new real estate property

#### PUT /realEstate/{id}
- Updates an existing real estate property

#### DELETE /realEstate/{id}
- Deletes an existing real estate property

#### GET /realEstate/search
- Executes an advanced search for real estate properties

## Users
Endpoints associated with users

#### POST /users/cadastro
- Creates a new user

#### POST /users/administradores
- Creates a new administrator

#### DELETE /users/usuarios/{id}
- Deletes a non-administrator user
  - Parameters:
    - `id`: *(string)* ID of the user to be deleted (path)
  - Responses:
    - `200`: User successfully deleted
    - `403`: Access denied or user is not an administrator
    - `404`: User not found or is an administrator
    - `500`: Internal server error

#### PUT /users/usuarios/{id}
- Updates user data

#### POST /users/login
- Handles user authentication

## Visits
Endpoints related to visits

#### GET /visits
- Retrieves a paginated list of all visits

#### POST /visits
- Creates a new visit

#### PUT /visits/{id}
- Updates an existing visit

#### DELETE /visits/{id}
- Deletes an existing visit
