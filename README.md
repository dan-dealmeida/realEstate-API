# API Endpoints

## Favorites
Endpoints related to favorites

#### GET /favorites
- Lists all favorites with pagination

#### POST /favorites
- Adds a new favorite

#### PUT /favorites/{id}
- Updates an existing favorite

#### DELETE /favorites/{id}
- Deletes an existing favorite

## RealEstate
Endpoints related to real estate

#### GET /realEstate
- Lists all real estate with pagination

#### POST /realEstate
- Creates a new real estate property

#### PUT /realEstate/{id}
- Updates an existing real estate property

#### DELETE /realEstate/{id}
- Deletes an existing real estate property

#### GET /realEstate/search
- Performs advanced search for real estate properties

## Users
Endpoints related to users

#### POST /users/cadastro
- Creates a new user

#### POST /users/administradores
- Creates a new administrator

#### DELETE /users/usuarios/{id}
- Deletes a non-administrator user
  - Parameters:
    - `id`: *(string)* ID of the user to be deleted (path)
  - Responses:
    - `200`: User successfully deleted
    - `403`: Access denied or user is not an administrator
    - `404`: User not found or is an administrator
    - `500`: Internal server error

#### PUT /users/usuarios/{id}
- Updates user data

#### POST /users/login
- User authentication

#### POST /visits

#### PUT /visits/{id}
- Updates an existing visit

#### DELETE /visits/{id}
